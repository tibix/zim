====== TAG55 - Symbol ======
Created Wednesday 08 June 2016

Ticker symbol. Common, "human understood" representation of the security. 
SecurityID ([[TAG48 - SecurityID|48]]) value can be specified if no symbol exists (e.g. non-exchange traded Collective Investment Vehicles)
Use **"[N/A]"** for products which do not have a symbol.
